    安装使用与功能菜单¶

    此页将帮助您安装与初始化Ankali。

    安装至Android¶

    进入Ankali的适合的项目仓库:

    • github

    • gitee

    找到Download AnKali APK标题项，选择适合你的下载方式下载安装至你的手机上。

    初始化¶

    进入程序后将解压kali镜像文件，此过程需要等待。 解压后需要您选择你的功能菜单语言，之后可能需要您填写用于初始化配置的信息。目前仅支持语言English与简体中文。

    完成初始化后，会出现"Function Menu"，此时的kali Linux环境已经安装至您的手机上了。 🎉🎉🎉🎉🎉

    功能菜单说明¶

    Enter Kali CLI(进入 Kali CLI)¶

    在功能菜单上输入数字 "1" 回车可登录kali命令行界面，初次使用可能需要时间。

    可以使用命令cat /etc/*release可用于显示系统发行版的详细信息。

    Set root account password(设置root帐户密码)¶

    对应命令passwd，用于修改root账号的密码。

    Configure system language(配置系统语言)¶

    用于配置切换kali Linux环境中的系统语言，支持约72个系统语言(可能部分语言无法切换)。可用于部分软件包或桌面环境的语言修改。

    Graphical Desktop GUI and VNC Service(图形桌面GUI和VNC服务)¶

    显示关于图形桌面环境与VNC服务的功能，包括于:

    • Install GUI Graphical Desktop(安装GUI图形桌面)
    • uninstall GUI Graphical Desktop(卸载GUI图形桌面)
    • VNC Service(VNC服务)

    Special Key(特殊键)¶

    用于提示组合键映射的特殊键的信息，可以参考特殊键的使用

    Start Tmoe(启动Tmoe)¶

    该功能需要在桌面环境中执行，用于启动tmoe菜单。了解tmoe

    Start Katoolin(启动Katoolin)¶

    用于启动Katoolin工具的使用。了解Katoolin

    Settings(设置)¶

    用于配置kali Linux与功能菜单的设置。

    install the basic default toolset(安装最基本的默认工具集)¶

    如果你想要在kali Linux安装最基本的默认工具集(这通常是kali系统自带的)，可以在功能菜单使用此功能将安装最基本的信息安全工具，它也代表 了Kali Linux 默认安装的软件集合。但是要注意的是这也将会占用大约8GB的手机磁盘空间。如果你还想安装更多工具集，可以参考体验基本完整的kali工具集

    功能菜单日志文件¶

    功能菜单的日志文件位于 /.kali-config/log/ 目录中，文件名以 %Y-%m-%d.log格式为命名。如果想要清除日志文件可以参考 清楚日志文件。
