    Ankali是什么¶

    Ankali是一款在Android设备上利用Proot技术在允许没有Root的环境上模拟较为完整但受限的kali Linux环境的Android应用程序。Ankali支持一键安装kali Linux(rolling)环境，无需如何复杂的操作，无需借用任何第三方Android应用程序。在中文上的Ankali可以被称为安卡利。

    Ankali的使用范围¶

    Ankali是模拟kali Linux环境的，他的使用范围是基于kali Linux的，只不过由于限制如无Root导致部分kali工具无法正常使用。使用范围部分包括:

    • 渗透测试: Ankali模拟的Kali Linux 环境包含了大量的安全工具，用于评估计算机网络和系统的安全性。

    • 数字取证：kali Linux提供了许多用于取证分析的工具，帮助安全专家在计算机系统、移动设备和网络中收集、分析和保护数字证据。但是Ankali模拟环境中可能会有部分限制与挑战。

    • 逆向工程：Ankali模拟的 Kali Linux 环境也提供了逆向工程工具。

    使用门槛¶

    Ankali是一款简单的应用程序，主要的门槛:

    • 内存要求：此应用程序会在初始使用时提取卡利图像文件，提取后将占用大约 2.5GB 的手机存储空间，请确保您的设备有足够的空间内存，甚至更多。

    • 设备要求：Ankali适配Android7.0及以上版本，最理想的可能会是Android11。如果您的Android版本过低或过高，或者您的设备不支持，则可能导致包括但不限于软件崩溃，镜像文件解压失败等问题。请确保您的设备支持此应用程序。

    • 构架要求: Ankali需要运行在aarch64(arm64)构架上的Android设备上，Linux内核版本的要求未知，请确保您的设备属于该构架。

    • 软件条约: 初次使用请详细阅读软件的隐私条约与免责声明

    • 合法使用: 在Ankali模拟的kali Linux环境支持安装非常多的各个领域的安全工具，使用此类工具需要在获得明确授权的情况下，对于组织的网络、系统或应用程序进行安全评估。这是 Ankali 最常见的合法用途之一。
