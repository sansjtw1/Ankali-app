    kali Linux桌面环境¶

    此页将帮助您在Ankali上使用kali Linux桌面环境。主要包括桌面环境的安装，桌面的删除，VNC的安装使用。

    Install GUI Graphical Desktop(安装GUI图形桌面)¶

    功能位于功能菜单Graphical Desktop GUI and VNC Service(图形桌面GUI和VNC服务)当中。Install GUI Graphical Desktop是一个选择菜单且其中提供7个桌面环境的自动化安装，根据自行的选择输入对应的桌面环境进行安装。

    安装图形桌面环境需要时间，请保持网络良好，存储空间充足的情况下安装。

    在安装桌面环境前建议先配置kali系统语言，当桌面安装时会自动根据你的系统语言配置您的桌面语言。

    在安装环境时建议XFCE桌面，XFCE桌面环境是一个轻量级的桌面环境，此桌面环境在Ankali中会带来高效的体验。

    在安装桌面环境的过程可能会出现参数配置界面TUI，可以根据自行要求选择，通过特殊键去操作该界面。当然可以在考虑后的情况下直接回车默认选择即可。

    VNC Service(VNC服务)¶

    当你的桌面环境成功安装至您的kali后，可以借助VNC服务通过启动VNC服务远程启动管理你的桌面。

    VNC全称Virtual Network Computing，是一种远程桌面软件协议，它允许用户远程控制另一台计算机的桌面环境。VNC工作原理是通过网络传输键盘和鼠标事件，并将远程计算机的屏幕图像传回给用户，从而实现远程操作。

    VNC Service功能位于功能菜单Graphical Desktop GUI and VNC Service(图形桌面GUI和VNC服务)当中。VNC Service主要用于

    • VNC服务的安装
    • VNC服务的配置
    • VNC服务的启动
    • 桌面GUI的启动

    而VNC Service菜单的上方则是配置信息:

    • VNC Service(VNC服务) 是否安装
    • NOVNC 是否安装
    • 显示 lnstalled desktop environment(安装的桌面环境)
    • VNC Password(VNC密码) 是否设置

    Install VNC Service(安装VNC服务)¶

    该功能用于安装VNC Service，如果你想在CLI手动安装它

    sudo apt update
sudo apt install -y tigervnc-standalone-server xinit

    Install NOVNC Service(安装NOVNC服务)¶

    该功能用于安装NOVNC Service，如果你想在CLI手动安装它

    sudo apt update
sudo apt install novnc websockify -y

    Configure VNC Password(配置VNC密码)¶

    当你的VNC服务已安装至kali中时，需要您使用该功能配置您的VNC访问密码。因为为了保证远程连接的安全性，这是必须的。如果你想在CLI手动设置它

    vncpasswd

    Start the graphical desktop and VNC service directly(直接启动图形桌面和VNC服务)¶

    当你已经完成以下需求后，即可使用该功能启动您的VNC服务与图形桌面:

    • 图形桌面已安装
    • VNC服务已安装
    • VNC密码已设置

    可以查看表格上的配置信息确认

    当您的Start the graphical desktop and VNC service directly启动后，会出现连接此VNC的IP地址与连接端口。

    Start VNC + NOVNC services(启动VNC + NOVNC服务)¶

    NOVNC 是一个基于 Web 的 VNC 客户端，它允许用户通过标准的网页浏览器来访问和控制远程计算机上的 VNC 服务器。

    当你已经完成以下需求后，即可使用该功能启动服务

    • 图形桌面已安装
    • VNC服务已安装
    • NOVNC已安装
    • VNC密码已设置

    可以查看表格上的配置信息确认

    当您的Start VNC + NOVNC services启动后，会出现对应连接的VNC连接端口与Novnc的URL链接。

    连接 VNC 服务¶

    当你启动Start VNC + NOVNC services后，可以通过VNC客户端连接它。

    获取VNC客户端

    当您的Start the graphical desktop and VNC service directly启动后，会出现连接此VNC的IP地址与连接端口。

    在客户端输入连接的IP地址与端口并连接它，RVNC Viewer为例

    连接 NOVNC 服务¶

    当你启动Start VNC + NOVNC services后，可以通过浏览器访问NOVNC的URL并远程控制VNC服务。

    当您的Start VNC + NOVNC services启动后，会出现对应连接的VNC连接端口与Novnc的URL链接。 在浏览器打开这个URL 点击Connect输入密码即可连接你的VNC服务。

    uninstall GUI Graphical Desktop(卸载GUI图形桌面)¶

    功能位于功能菜单Graphical Desktop GUI and VNC Service(图形桌面GUI和VNC服务)当中。uninstall GUI Graphical Desktop功能会自动检测已安装的桌面并列出菜单，你可以选择需要删除的已安装的桌面进行卸载桌面软件包
